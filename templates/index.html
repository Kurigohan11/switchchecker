📋 Шпаргалка по командам

Звёздочка * = номер порта. В скобках — альтернативные варианты обозначения.

🔵 Linksys 24-port
▼
КомандаДействие
sh int stПроверка всех портов
sh mac add eth e* (g*)Проверка MAC-адреса порта
test c t e* (g*)Тест кабеля порта
sh int cou eth e* (g*)Проверка ошибок на порту
clear cou eth e* (g*)Очистка ошибок с порта
sh ip dhcp snooping bindingПроверка IP-адресов

💡 Порты: e* = ethernet (100M), g* = gigabit uplink

🔷 Cisco SF300 · MES1124MB rev.B · MES2324B 1G/10G
▼
КомандаДействие
sh int stПроверка всех портов
sh mac add int fa* (gi*)Проверка MAC-адреса порта
test cable-diagnostics tdr int fa* (gi*)Тест кабеля порта
sh int cou fa* (gi*)Проверка ошибок на порту
clear cou fa* (gi*)Очистка ошибок с порта
sh ip dhcp snooping bindingПроверка IP-адресов

💡 Порты отличаются в зависимости от свитча:
fa1, gi1, fa1/0/1, fa3/0/1, gi1/0/1, gi3/0/1
Выбирай тип порта в выпадающем списке при диагностике.

🔶 Eltex MES2424B AC · MES2424B 28port
▼
КомандаДействие
show interfaces staПроверка всех портов
sh mac int g0/*Проверка MAC-адреса порта

show int cou gi0/*Проверка ошибок на порту
clear cou gi0/*Очистка ошибок с порта

💡 MES2424B AC определяется по серийному номеру (начинается с ESF3).
MES2424B 28port — по имени устройства в промпте.
