# SwitchPro — Диагностика коммутаторов

Веб-приложение для удалённой диагностики сетевых коммутаторов по SSH.

## Поддерживаемые модели
- Cisco SF300 24/48 (10/100)
- MES 2424B AC / 28 port
- MES 1124MB Rev.B
- Linksys 24-port
- MES 2324B (1G/10G)

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python app.py
```

Откройте браузер: http://localhost:5000

## Файл паролей

В файле `passwords.txt` (в той же папке) хранится пароль — одна строка.
Путь к файлу можно изменить в интерфейсе.

## Возможности

| Кнопка | Команда |
|--------|---------|
| Все порты — статус | `show interfaces status` |
| Порт — информация | `show interfaces ... {port}` |
| Тест кабеля (TDR) | `test copper-tdr / test cable-diagnostics` |
| MAC-адреса на порту | `show mac address-table interface ...` |
| Ошибки на порту | `show interfaces ... counters` |
| Сбросить счётчики | `clear counters ...` |
| Своя команда | произвольная команда |

## Логика определения модели

При подключении программа выполняет `show version` и анализирует вывод.
Если модель определена — автоматически подбираются правильные команды.
